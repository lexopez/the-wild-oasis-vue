# 🌲 The Wild Oasis

A modern **Vue js** web application for discovering and booking luxury cabins in nature. _The Wild Oasis_ blends a calm, premium aesthetic with fast performance, server-first rendering, and a clean user experience.

🔗 **Live Demo:** [https://wild-oasis-reactjs-app.netlify.app](https://wild-oasis-reactjs-app.netlify.app/dashboard)

---

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
